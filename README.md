# Data Structures & Algorithms in C

This repository contains three core systems built in C as part of CSCA48 at the University of Toronto.

Each project focuses on implementing fundamental data structures from scratch and applying them to real-world-style problems.

---

## Projects

### 1. Movie Review Database (Linked Lists)
A fully functional in-memory database built using singly linked lists and compound data types. Supports insertion, deletion, search, sorting, and analytics over movie and cast data.

### 2. Music Sequencer (Binary Search Trees)
A time-ordered music engine implemented using a Binary Search Tree, where each node represents a musical note. Supports playback ordering, traversal, reversal, and harmonization.

### 3. Food Networks (Graphs + Recursion)
A graph-based ingredient recommendation system using adjacency matrices and recursive traversal to explore ingredient relationships and substitutions.

---

## Technical Highlights
- Manual memory management using `malloc` / `free`
- Recursive algorithms (trees and graphs)
- Implementation of BST deletion (all cases)
- Graph traversal with constraints and depth limits
- Data modeling using structs and pointers
- Testing and debugging of non-trivial systems
